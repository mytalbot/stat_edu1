# stat_edu1
Contains working material for a basic course in statistics using the software GraphPad Prism.

The repository contains 3 files:
* a PDF with a selection of statistical tests in form of a decision tree
* an MS-Excel file with a selection of statistical exercises
* the encrypted slides (PDF) of the corresponding course
